IPAC CONTROLLER KEY LAYOUT:
------------------------------------------------------------
1P up    : UP                             2P up    : NUM8
1P down  : DOWN                           2P down  : NUM2
1P left  : LEFT                           2P left  : NUM4
1P right : RIGHT                          2P right : NUM6
1P but1  : RIGH CONTROL                   2P but1  : Num5
1P but2  : LEFT ALT                       2P but2  : S
1P but3  : SPACE                          2P but3  : Q
1P but4  : LEFT SHIFT                     2P but4  : W
1P but5  : Z                              2P but5  : E
1P but6  : X                              2P but6  : R
1P Start : 1                              2P start : 2
1P coin  : 5                              2P coin  : 6
 - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
1P SHIFTED up    : M
1P SHIFTED down  : P
1P SHIFTED left  : Volume Down
1P SHIFTED right : Volume Up
1p SHIFTED but1  : Esc
1p SHIFTED but2  : F12
1p SHIFTED but3  : Macro (F12 + Q)
